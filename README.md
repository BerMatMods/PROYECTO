## 📱 Descripción del Proyecto

**YAPE Pro** es una aplicación de pagos móviles desarrollada con **tecnologías avanzadas de desarrollo web moderno**, replicando la experiencia de usuario de la app oficial con funcionalidades mejoradas y una arquitectura optimizada para rendimiento y escalabilidad.

### 🔄 Flujo de Datos

```javascript
// Patrón de gestión de estado implementado
const AppState = {
```

## 🛠️ Configuración y Personalización

### 🎨 Personalización de Estilo

Edita las variables CSS en el `<head>` para adaptar la app a tu marca:

```css
:root {
  /* Cambia los colores principales */
  --yape-purple: #TU_COLOR_PRIMARIO;
  --yape-turquoise: #TU_COLOR_SECUNDARIO;
  
  /* Ajusta sombras y efectos */
  --shadow: 0 4px 20px rgba(0,0,0,0.15);
  
  /* Personaliza tipografía */
  font-family: 'TuFuentePreferida', sans-serif;
}
```

### ⚙️ Configuración de Datos Iniciales

Modifica el objeto `userData` en la sección `<script>`:

```javascript
let userData = {
  name: 'Tu Nombre Completo',
  nickname: 'Tu Apodo o Alias',
  email: 'tu.correo@ejemplo.com',
  phone: '+51 9XX XXX XXX',
  photo: 'URL_de_tu_foto_o_base64',
  balance: 1000.00,  // Saldo inicial
  balanceVisible: false  // Ocultar saldo por defecto
};
```

### 🔊 Configuración de Audio

Para cambiar el sonido de notificación:

```html
<!-- Reemplaza el src con tu archivo de audio -->
<audio id="notificationSound" src="ruta/a/tu-audio.mp3" preload="auto"></audio>
```

## 📚 Documentación Técnica Adicional

### 🗂️ Estructura de Componentes

```
index.html
├── <head>
│   ├── Meta tags para PWA y mobile
│   ├── CSS Variables Design System
│   ├── Animaciones y efectos CSS
│   └── Responsive & Safe Area Styles
│
├── <body>
│   ├── Header (Top Bar + Menú)
│   ├── Content (Services + Carousel + Balance)
│   ├── Bottom Actions (QR + Yapear)
│   ├── Modals (Send, Profile, Balance, Info, ClearHistory)
│   ├── Screens (Confirm, Receipt, Scan)
│   ├── Effects (Sparkles, Notifications)
│   └── Audio Elements
│
└── <script>
    ├── State Management (userData, movements)
    ├── UI Rendering (updateUI, renderMovements)
    ├── Event Handlers (toggleMenu, confirmSend, etc.)
    ├── Persistence Layer (loadData, saveData)
    ├── Utilities (showNotification, showSparkles)
    └── Initialization (DOMContentLoaded, listeners)
```

### 🔄 Ciclo de Vida de la Aplicación

```javascript
// 1. Carga Inicial
window.load → loadData() → updateUI() → bindEvents()

// 2. Interacción del Usuario
userClick → validateInput() → processAction() → saveData() → updateUI() → showFeedback()

// 3. Persistencia Automática
onDataChange → debounce(500ms) → localStorage.setItem() → syncState()

// 4. Actualización de UI
stateChange → requestAnimationFrame() → reRenderComponents() → applyTransitions()
```
